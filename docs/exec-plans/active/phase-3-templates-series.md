# Execution Plan: Phase 3 — Templates & Series

**Status:** planned
**Depends on:** Phase 2 complete

## Goals

Add plot templates, series management, and deeper Google Docs sync.

## Task Breakdown

### 1. Plot Templates
- [ ] Database: templates table with JSONB structure
- [ ] Built-in templates:
  - [ ] Three-Act Structure
  - [ ] Hero's Journey (Joseph Campbell)
  - [ ] Save the Cat! (Blake Snyder)
  - [ ] Story Circle (Dan Harmon)
  - [ ] Romancing the Beat
  - [ ] Snowflake Method
  - [ ] 7-Point Plot Structure
  - [ ] 12-Chapter Mystery
  - [ ] Children's Book Formula
  - [ ] Freytag's Pyramid
- [ ] Template browser: preview and apply to a book
- [ ] Apply template: creates plotlines + chapters + placeholder scenes
- [ ] Custom templates: save current book structure as a template
- [ ] Character profile templates (15+ built-in)

### 2. Series Management
- [ ] Project type: standalone vs series
- [ ] Series project: multiple books with cover images and sort order
- [ ] Series-level timeline view (spans across books)
- [ ] Shared story bible (characters, places, notes, tags span all books)
- [ ] Book selector in navigation
- [ ] Series overview dashboard

### 3. Deeper Google Docs Sync
- [ ] Sync word count on a schedule (not just on-demand)
- [ ] "Back to plotamour" link in Doc header
- [ ] Update Doc context header when scene details change
- [ ] Writing progress dashboard per book (total words, scenes completed)
- [ ] Export outline to a single Google Doc (formatted)

## Definition of Done

- Writers can start a project from a template instead of a blank slate
- Series authors can manage multi-book projects with shared world-building
- Google Docs integration feels seamless with progress tracking
