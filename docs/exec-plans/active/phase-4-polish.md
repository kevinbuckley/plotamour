# Execution Plan: Phase 4 — Polish & Monetization

**Status:** planned
**Depends on:** Phase 3 complete

## Goals

Export features, ads integration, and UX polish.

## Task Breakdown

### 1. Export
- [ ] Export outline to PDF
- [ ] Export outline to Word (.docx)
- [ ] Export outline to Google Docs (single formatted document)
- [ ] Export character profiles to PDF
- [ ] Export full project summary (outline + characters + places)

### 2. Ads Integration
- [ ] Ad slot in sidebar (below navigation)
- [ ] Ad slot on dashboard (between projects)
- [ ] Ad provider integration (Google AdSense or similar)
- [ ] Respect: no ads inside the timeline or writing flow
- [ ] Ad-free option for future premium tier

### 3. UX Polish
- [ ] Onboarding flow: guided tour for first-time users
- [ ] Keyboard shortcuts (N: new scene, E: edit, D: delete, W: write)
- [ ] Undo/redo for drag-and-drop operations
- [ ] Trash/archive view for soft-deleted items
- [ ] Loading states and skeleton screens
- [ ] Error boundaries and graceful error messages
- [ ] Empty states with helpful prompts

### 4. Performance
- [ ] Optimize timeline rendering for large projects
- [ ] Lazy load story bible tabs
- [ ] Image optimization for character/place/book covers
- [ ] Database query optimization with proper indexes

### 5. Accessibility
- [ ] Keyboard navigation for timeline grid
- [ ] Screen reader labels for scene cards
- [ ] Color contrast compliance
- [ ] Focus management for modals and panels

## Definition of Done

- App feels polished and professional
- Export works for the most common use cases
- Ads are present but not intrusive
- Performance is smooth for projects up to 50 chapters × 10 plotlines
