# Product Sense — plotamour

## Who We Build For

**Hobby novelists.** People with a day job, a family, and a story burning inside them. They write in the mornings, on weekends, during lunch breaks. They don't have MFAs or literary agents. They have Google Docs and ambition.

### User Persona: "Weekend Writer"
- Writes fiction as a passion project (novels, novellas, short story collections)
- Uses Google Docs for writing (familiar, free, accessible)
- Has tried outlining in spreadsheets, notebooks, or Notion — none felt right
- Wants to see the shape of their story without learning complex software
- All genres: romance, mystery, fantasy, sci-fi, literary fiction, etc.

### What They Need
1. A visual way to see their story's structure (not just a list)
2. Easy back-and-forth between planning and writing
3. A place to track characters, places, and story details
4. Something that doesn't overwhelm them with features on day one

### What They Don't Need
- Publishing tools
- Manuscript formatting
- AI writing assistance (not yet)
- Collaboration with editors/co-authors (not in v1)
- Mobile app (desktop writers)

## Product Principles

### 1. Plan ↔ Write is one motion
The gap between "I know what happens next" and "I'm writing it" should be one click. The Google Docs integration exists to make this seamless.

### 2. Show, don't tell
The timeline view communicates more in a glance than any bullet list. Visual planning is our superpower.

### 3. Grow with the writer
Start simple: one plotline, a few chapters, a few scenes. As the story gets complex, introduce plotlines, tags, character tracking. Never force complexity on a new user.

### 4. Your words are sacred
We never delete a user's writing. Soft deletes everywhere. Google Docs are never removed, only unlinked. If we lose trust, we lose everything.

### 5. Open source, open heart
plotamour is free and open source. Revenue comes from ads, not from gating features. Every writer deserves good tools regardless of budget.

## Metrics We Care About

| Metric | Why |
|---|---|
| Projects created | Are people starting? |
| Scenes with linked Docs | Are people using the core loop? |
| Return visits (7-day) | Are people coming back? |
| Word count across Docs | Are people actually writing? |
| Time in timeline view | Is visual planning working? |

## Competitive Landscape

| Tool | Strength | Our Edge |
|---|---|---|
| Plottr | Visual timeline, templates | Google Docs round-trip, free + open source |
| Scrivener | Full manuscript management | Simpler, web-based, Google Docs native |
| Notion | Flexible databases | Purpose-built for fiction, timeline view |
| Google Docs alone | Writing | No visual structure or story bible |
| Spreadsheets | Flexible | Purpose-built UI, not a hack |
